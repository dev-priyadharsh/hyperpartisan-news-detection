# Hyperpartisan News Detection (CW1)

This repository contains a Colab notebook for classifying news articles as hyperpartisan or non-hyperpartisan using classical machine learning models (Logistic Regression, SVM, Random Forest, Naive Bayes) on the SemEval-2019 dataset.

## Usage

- Open `hyperpartisan_detection_CW1.ipynb` in [Google Colab](https://colab.research.google.com/) or Jupyter.
- Upload the required XML files in the working directory.
- Run all cells.
- Results and plots will appear in the output cells.
